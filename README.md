# はじめに
山佐時計計器株式会社製
[USBポケット万歩計TH-850](http://www.yamasa-tokei.co.jp/seihin/support_service/th_850.html)
から情報を読み出すライブラリです。

独自に解析したUSBコマンドから作成したライブラリであり、
内容については山佐時計計器株式会社とは全く関係ありません。

また、本ライブラリを使用することで、万歩計に機能に損傷が発生しても、
一切補償はない旨ご了承の上利用ください。
